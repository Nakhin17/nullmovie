คำอธิบายเรื่องย่อรายละเอียดของภาพยนตร์เรื่องนี้...

หมายเหตุสำหรับการเล่นวิดีโอจากเครื่องคอมพิวเตอร์ตนเอง

หากใช้ลิงก์ในเครื่องตนเอง (เช่น http://localhost:...) ต้องเปิดใช้เซิร์ฟเวอร์หลังบ้านในคอมพิวเตอร์ของคุณตามคู่มือระบบหลังบ้านก่อน จึงจะสามารถเล่นหนังได้อย่างลื่นไหล

เพิ่ม/แก้ไขหนังเข้าระบบ NullMovie

สามารถใช้ URL ภาพยนตร์จากอินเทอร์เน็ตทั่วไป หรือนำ URL จาก เซิร์ฟเวอร์ในคอมพิวเตอร์ของคุณ ด้านขวามือมาใส่

วิธีทำให้คอมฯ ตัวเองเป็น Server สตรีมมิ่งหนัง

คุณสามารถเปิดไฟล์วิดีโอหนังที่อยู่ในฮาร์ดดิสก์เครื่องคอมพิวเตอร์คุณ แล้วสตรีมเปิดเล่นผ่านเว็บไซต์นี้ได้ฟรี! โดยทำตามขั้นตอนสั้นๆ ดังนี้:

1

สร้างโฟลเดอร์เก็บหนังในเครื่อง

เช่น สร้างโฟลเดอร์ชื่อ C:\my-movies แล้วเอาไฟล์หนังที่เป็นนามสกุล .mp4 ไปใส่ไว้

2

เปิดระบบ Web Server ด้วย Python (ง่ายที่สุด)

ในคอมพิวเตอร์ของคุณ ให้เปิดโปรแกรม Terminal (macOS) หรือ Command Prompt (Windows) แล้วรันโค้ดด้านล่างนี้:

cd C:\my-movies
python -m http.server 8000

(หมายเหตุ: เครื่องคุณต้องติดตั้ง Python ก่อน หากต้องการดึงไฟล์หนัง)

3

นำลิงก์ภาพยนตร์มากรอก

หากในโฟลเดอร์มีไฟล์หนังชื่อ matrix.mp4 ช่องวิดีโอฝั่งซ้ายให้กรอกเป็น:

http://localhost:8000/matrix.mp4
